# 🍽️ Bangalore Zomato Restaurant Analysis

## 📌 Project Overview

This is an End-to-End Data Analyst project that analyzes Bangalore's Zomato restaurant dataset using **Python, PostgreSQL, and Power BI**.

The objective of this project is to clean raw restaurant data, perform SQL analysis, build an interactive Power BI dashboard, and generate business insights for better decision-making.

## 🔍 Project Workflow

### 1. Data Cleaning (Python)

- Removed duplicate records
- Handled missing values
- Converted rating into numeric values
- Converted cost into numeric format
- Cleaned restaurant names and categories

---

### 2. Exploratory Data Analysis

Performed analysis on:

- Restaurant distribution
- Cuisine popularity
- Restaurant types
- Online ordering
- Table booking
- Ratings
- Cost analysis
- Votes analysis

---

### 3. PostgreSQL Analysis

Performed SQL analysis using:

- Aggregate Functions
- GROUP BY
- HAVING
- ORDER BY
- LIMIT
- Subqueries
- Filtering
- Distinct values

More than **20 SQL queries** were executed for business insights.

---

### 4. Power BI Dashboard

Interactive dashboard includes:

- KPI Cards
- Top 10 Locations
- Top 10 Cuisines
- Restaurant Type Distribution
- Top Restaurants by Votes
- Average Cost by Restaurant Type
- Online Order Analysis
- Table Booking Analysis
- Interactive Filters (Location, Restaurant Type, Cuisine)

---

## 📈 Key Insights

- BTM has the highest number of restaurants.
- North Indian cuisine is the most popular cuisine.
- Casual Dining is the most common restaurant type.
- More than 60% of restaurants support online ordering.
- Only around 14% provide table booking.
- Premium restaurant types have the highest average cost.
